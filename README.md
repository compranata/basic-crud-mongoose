# basic-crud-mongoose

## Environment
- Node.js v10.0
- Expressjs v4.16
- Mongod v4.0.3
- Mongoose v5.3.13

## Install
```
$ cd ./project
$ express --view=pug basic-crud-mongoose
$ cd ./basic-crud-mongoose
$ npm install
$ npm install --save mongoose

// for Paging Branch
$ npm install --save express-paginate mongoose-paginate
```
